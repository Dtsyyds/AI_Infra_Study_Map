# 0004. 寻找两个正序数组的中位数

## 题目链接：

https://leetcode.cn/problems/median-of-two-sorted-arrays/solutions/3987174/tian-chong-xin-shu-zu-qian-xian-pai-hao-gtm8r/

## 题目类型：

- 暴力搬移数组
- 在生成新数组前先排好序再插入

## 题解

- 暴力解法

直接将两个数组搬移到一起，然后排序，求解中位数

- 相对暴力

按大小顺序搬移进新数组，由于两个数组的长度已知，因此中位数对应的两个数组的下标之和也是已知的。然后求解中位数

## 官方题解 - 二分查找

可以将这道题目转化成寻找第 k 小的数的问题，然后使用二分查找来解决。

不断比较 A[k/2 - 1] 和 B[k/2 - 1], 就可以确定第 k/2 个数是在 A 还是 B 中，不断更新迭代，直到找到第 k 小的数。

这道题的本质不是直接找中位数，而是：

> 找到一个位置，把两个数组整体分成左右两部分，使得左边所有数都小于等于右边所有数。

也就是：

```text
左半部分 | 右半部分
```

我们希望满足：

```text
左边元素数量 == 右边元素数量
```

或者：

```text
左边元素数量比右边元素数量多 1
```

为什么允许左边多一个？

因为当总长度是奇数时，中位数就是左半部分的最大值。

---

## 三、在两个数组中分别切一刀

假设：

```cpp
nums1 = [1, 3, 8]
nums2 = [2, 7, 10, 11]
```

我们不真正合并数组，而是在两个数组中各切一刀：

```text
nums1: [1, 3] | [8]
nums2: [2, 7] | [10, 11]
```

此时左半部分是：

```cpp
[1, 3, 2, 7]
```

右半部分是：

```cpp
[8, 10, 11]
```

如果左半部分所有元素都小于等于右半部分所有元素，那么这个切分就是正确的。

---

## 四、只需要检查边界元素

由于 `nums1` 和 `nums2` 本身都是有序数组，所以不用检查左边和右边的所有元素。

只需要检查分割线附近的四个数：

```text
nums1 左边最大值：Aleft
nums1 右边最小值：Aright

nums2 左边最大值：Bleft
nums2 右边最小值：Bright
```

正确切分需要满足：

```cpp
Aleft <= Bright && Bleft <= Aright
```

只要这个条件成立，就说明：

```text
左边所有元素 <= 右边所有元素
```

---

## 五、为什么可以用二分查找？

我们只在较短的数组上进行二分。

假设：

```cpp
i = nums1 的切分位置
j = nums2 的切分位置
```

两个数组左半部分的总长度应该是：

```cpp
leftSize = (m + n + 1) / 2
```

因此：

```cpp
j = leftSize - i
```

也就是说：

> 只要确定了 `nums1` 的切分位置 `i`，`nums2` 的切分位置 `j` 就自动确定了。

所以我们只需要在 `nums1` 中二分查找 `i`。

---

## 六、二分查找时如何调整方向？

当前切分如下：

```text
nums1: ... Aleft | Aright ...
nums2: ... Bleft | Bright ...
```

### 情况一：切分正确

如果：

```cpp
Aleft <= Bright && Bleft <= Aright
```

说明当前切分合法，可以计算中位数。

---

### 情况二：`Aleft > Bright`

说明 `nums1` 左边拿得太多了。

因为 `nums1` 左边的最大值，竟然比 `nums2` 右边的最小值还大。

此时应该让 `nums1` 的切分位置向左移动：

```cpp
right = i - 1;
```

---

### 情况三：`Bleft > Aright`

说明 `nums1` 左边拿得太少了。

因为 `nums2` 左边的最大值，比 `nums1` 右边的最小值还大。

此时应该让 `nums1` 的切分位置向右移动：

```cpp
left = i + 1;
```

---

## 七、找到正确切分后如何返回答案？

找到合法切分后：

```cpp
leftMax = max(Aleft, Bleft);
rightMin = min(Aright, Bright);
```

如果总长度是奇数：

```cpp
return leftMax;
```

如果总长度是偶数：

```cpp
return (leftMax + rightMin) / 2.0;
```
