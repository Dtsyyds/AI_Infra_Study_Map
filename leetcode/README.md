# LeetCode 刷题计划与题单

> **半年目标**：累计完成 120-150 道题。重点不是题量，而是能独立写出、能复盘、能讲清楚。

---

## 📋 刷题规则

1. **独立思考**：每题先独立思考 15 分钟，不轻易看题解。
2. **题解复写**：没思路再看题解，看完后关掉页面，自己重新写一遍。
3. **复盘记录**：每题记录题型、核心思路、错误原因（见下方记录模板）。
4. **隔日复写**：第二天复写前一天没一次 AC 的题，直到流畅写出。

# 💡 刷题建议与复习策略

1. **按题型集中刷**：不要每天随机选题。把同类题放在一起刷，更容易总结出固定模板。
2. **核心母题优先**：题单中标注为“模板题”或“核心模式”的题目是基础，先做透这些，变种题自然能举一反三。
3. **主攻中等难度**：AI 岗位面试算法题以 LeetCode **Medium** 难度为主，不必在偏题怪题上花太多时间。
4. **严格复盘**：每天刷完题后，务必按模板填写记录（尤其关注“错误原因”）。
5. **隔日复写**：针对没一次 AC 的题，第二天先不看答案重新写，直到一次通过为止。
6. **总题量规划**：
   - 阶段一：约 40-50 题
   - 阶段二：约 40-50 题
   - 阶段三：约 30-40 题
   - **合计：约 120-150 题** ✅
