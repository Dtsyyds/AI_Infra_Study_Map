# LeetCode 刷题记录

## 目标

半年累计完成 120-150 道题，重点不是题量，而是能独立写出、能复盘、能讲清楚。

## 刷题规则

1. 每题先独立思考 15 分钟。
2. 没思路再看题解。
3. 看完题解后关掉页面，自己重新写。
4. 每题记录：题型、核心思路、错误原因。
5. 第二天复写前一天没一次 AC 的题。

## 阶段路线

| 阶段 | 时间 | 题型 |
|---|---|---|
| 阶段 1 | 第 1-2 月 | 数组、哈希表、字符串、双指针、栈、队列、链表 |
| 阶段 2 | 第 3-4 月 | 二分、滑动窗口、二叉树、DFS/BFS、堆、回溯、贪心 |
| 阶段 3 | 第 5-6 月 | 图、并查集、拓扑排序、动态规划、单调栈、综合题 |
